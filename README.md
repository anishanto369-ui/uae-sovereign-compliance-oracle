# 🇦🇪 UAE Sovereign Compliance Oracle (v4.0)
**Consulting-Grade Regulatory Intelligence & GRC Engine for 2026**

## 🚀 The Vision
In 2026, the UAE's regulatory landscape has reached a new level of complexity. Between the **VAT 5-year statute of limitations** (Federal Law No. 16 of 2025) and the **CBUAE April 2026 Proliferation Financing (PF)** mandates, manual auditing is no longer a viable risk strategy for high-stakes finance. 

The **Sovereign Compliance Oracle** is an enterprise-grade AI Architect designed to bridge the "Complexity Gap" for GCC institutions.

## 🏗️ Architectural Core
* **🛡️ Sovereign Privacy Layer:** Integrated middleware that redacts PII (Emirates IDs, Phone numbers) before any data leaves the local environment, enforcing **UAE Federal Decree Law No. 45**.
* **🏛️ Multi-Corpus Triangulation:** Autonomous search agents that cross-reference Federal Tax Authority (FTA) laws with Central Bank (CBUAE) 2026 guidelines.
* **⚖️ Explainable AI (XAI):** Delivers documented reasoning paths and risk ratings (Tax, AML, PF) instead of simple "chat" outputs.
* **🌍 Native Bilingual Intelligence:** Technical reporting generated simultaneously in English and Arabic for local and international C-suite alignment.

## 🛠️ Tech Stack
- **Engine:** Llama 3.3-70B (Groq LPU Inference)
- **Search:** Tavily Advanced Research API
- **UI:** Streamlit (GRC-Standard Dashboard)
- **Security:** Custom Regex-based PII Scrubber Middleware

## ⚙️ Implementation
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GROQ_API_KEY` and `TAVILY_API_KEY` to a `.env` file.
4. Launch the Oracle: `python3 -m streamlit run interface.py`
